    # 迭代器 Iterator

    TIP

    可迭代的值:Array、String、Map、Set

    const nums = ["zero", "one", "two"]; 
for (let num of nums) { 
    console.log(num)
}

    # of操作数必须是可迭代，这意味着如果是普通对象则无法进行迭代。如果数据结构类似于数组的形式，则可以借助 Array.from() 方法进行转换迭代。

    for (let item of Array.from(arrayLink)) { 
    console.log(item); 
}
